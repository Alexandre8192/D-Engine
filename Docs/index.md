# D-Engine Docs

This site is a navigation layer above the canonical handbook.
It is meant to help readers find the right architectural entry points faster
without creating a second source of truth.

## Source of truth

The canonical documentation is the repository root handbook:

- `../D-Engine_Handbook.md`

If a map or generated view disagrees with the handbook, treat that mismatch as a
documentation bug and fix it.

## What lives here

- `Handbook`: a generated copy of the root handbook for site navigation
- `Architecture`: curated maps plus generated landscape/container views
- `Historical snapshots`: milestone docs that may be stale and are kept for context
