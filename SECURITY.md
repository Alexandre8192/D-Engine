# Security Policy

## Supported Versions

`main` is the only actively supported branch for security fixes.

## Reporting a Vulnerability

Do not open a public issue for suspected vulnerabilities.

Report privately via:

- GitHub Security Advisory (preferred):
  `https://github.com/Alexandre8192/D-Engine/security/advisories/new`
- Or email: `alexandre.delangre9@gmail.com`

Please include:

- affected commit/tag/branch
- reproduction steps
- impact assessment
- suggested mitigation (if available)

## Response Targets

- Initial acknowledgment: within 72 hours
- Triage/update: within 7 days

If accepted, the fix is prepared on a private branch and disclosed after patch release.
